📖 ArX-AI User Manual
Your comprehensive guide to mastering the ultimate private AI workspace.

🧭 Left Sidebar Menu

Use the left sidebar buttons to move between the main ArX-AI workspaces. Each button opens a dedicated workflow area with its own controls, content and AI tools.

  • Chat: Start a conversation with your local AI models, type prompts, send messages, and enable Agent Mode for browser, file, or shell actions.
  • Deep Research: Run autonomous research jobs, search the web and arXiv, synthesise sources, and generate structured reports.
  • Model Arena: Compare two models side-by-side using the same prompt, vote blindly, and discover which model performs best.
  • Model Cookbook: Browse recommended local models, view hardware suggestions, and download models with one click.
  • Documents: Edit rich text documents, ask AI for rewrites, summaries, and grammar improvements, then save content directly into your workspace.
  • Notes & Tasks: Capture quick notes, manage todo lists, and let ArX extract action items from your chat or document content.
  • Email: Connect IMAP/SMTP accounts, draft replies, and let the AI help triage incoming messages.
  • Calendar: Review your schedule, create events, and manage tasks via CalDAV-compatible calendar integration.
  • User Manual: Opens this help page with step-by-step guides for every main feature.
  • Settings: Configure your local Ollama endpoint, API keys, integrations, and security options.

💬 Chat & Agent Mode

ArX allows you to chat with multiple local models via Ollama. By enabling Agent Mode (🤖), your AI gains superpowers:

  • 🌐 Web: Allows the agent to browse the internet via a private SearXNG instance.
  • 📎 Files: The agent can read and write files directly in your workspace.
  • 💻 Shell: The agent can execute sandboxed shell commands to interact with your system.
  • 🧠 Memory: Enables long-term vector database storage using ChromaDB to recall past conversations and documents.

🐝 Swarm Mode (Multi-Agent)

Toggle the Swarm Mode to activate a multi-agent workflow. When you send a prompt, a team of specialized agents will collaborate to solve it:

  • Researcher (🔍): Gathers information and creates an outline.
  • Writer (✍️): Drafts the content based on the outline.
  • Reviewer (🧐): Critiques the draft and outputs the final polished result.

🎙️ Voice Mode (Jarvis)

Click the 🎙️ Voice button to open the Jarvis overlay. A green interactive orb will appear.

Click the orb to start listening. Speak naturally, and click the orb again to stop listening and execute the command. The AI will reply with synthesized voice.

🐍 Python Code Interpreter

ArX includes an embedded WebAssembly Python interpreter (Pyodide). When the AI outputs a Python code block, you'll see a Run button next to it.

Clicking Run executes the code entirely in your browser. It supports matplotlib for inline graphs and charts! (Note: the first launch requires downloading ~20MB).

⚔️ Model Arena & Deep Research

Arena: Test two models blindly side-by-side. Vote on the best response and reveal the winner to track your favorite local models.

Deep Research: Provide a query, and the system will orchestrate an autonomous research task, pulling documents and summarizing them into a detailed Markdown report.

📝 Editor, Notes & Integrations

Use the built-in rich text Editor to draft documents with the help of AI suggestions.

Keep track of your ideas in Notes & Tasks.

Connect IMAP/CalDAV servers in the settings to let the AI triage your Email and organize your Calendar.

Reference: Buttons, Icons, and Controls

Sidebar

  • ＋ (New): Create a new chat/document/note depending on the active view.
  • × (Delete): Permanently removes the selected item from local storage. Use with caution.
  • Language selector: Changes basic UI labels (sidebar buttons, topbar title).

Topbar

  • Model Picker: Choose the model used for responses. Click the label to open the picker.
  • Agent (🤖): Toggles Agent Mode to allow web/file/shell tools (requires proper configuration in Settings).
  • Share: Export or share content from the current view.

Chat input

  • Enter: Send message. Shift+Enter inserts a newline.
  • 🎙️ Voice: Start voice dictation. Click again to stop and submit.
  • Token counter: Shows an approximate token usage for the current input.

Examples and Workflows

Draft an email: In Chat ask "Draft a professional reply to this email summarising the key points." Copy the suggested reply into Email or click "Draft reply" from the email AI bar.

Research to document: Use Deep Research to gather sources, then export the summary into Documents and refine with AI suggest.

Meeting notes: After a chat, extract action items by asking the assistant to "Extract action items" and then save them to Notes or convert to Tasks.

Troubleshooting & Tips

Voice not working? Ensure your browser supports the Web Speech API (Chrome/Edge). Grant microphone permissions and refresh.

Models unreachable? Check the Ollama endpoint in Settings and verify the server is running (default: http://localhost:11434).

Pyodide errors: First run downloads dependencies; wait a few seconds. If execution fails, open devtools to inspect console errors.

Reset local data: To permanently remove all saved chats, notes, documents, emails and events, clear app storage in Settings or run localStorage.removeItem('arx_state') in the browser console and reload the page.
